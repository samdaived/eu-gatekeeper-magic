# Welcome

This repository contains a Vite + React project.

## Local development

Requirements: Node.js and npm (or pnpm/yarn).

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

## Project tech

- Vite
- TypeScript
- React
- Tailwind CSS

## Deploy

Build for production with:

```bash
npm run build
```

Serve the production build with `npm run preview` or deploy to your preferred hosting provider.
